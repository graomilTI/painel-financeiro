Como subir:
1) Suba todos os arquivos mantendo as mesmas pastas.
2) Execute o SQL opcional para cadastrar o módulo App Gestor.
3) Acesse: https://grao1000.com.br/painel/gestor-app
4) No Android/Chrome: menu do navegador > Adicionar à tela inicial / Instalar app.

Primeira versão do app:
- Home mobile do gestor.
- OS em cards para celular.
- Indicação automática por distância real do Mapa Operacional.
- Ações rápidas: Aguardar, Atender, Finalizar.
- Ao atender, grava colaborador indicado e envia para Conferência.
- Módulos Programação, Hospedagem, Compras, Logística, Patrimônios e Contato Cliente abrem a tela web atual.

Observação:
O app usa cache local de alguns minutos para abrir mais rápido. O botão ↻ força atualização do banco.
